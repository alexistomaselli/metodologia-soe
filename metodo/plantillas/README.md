# Plantillas

Plantillas reutilizables para ejecutar tareas repetibles de la metodología.

> Las plantillas aparecen cuando una parte del método ya está suficientemente
> probada como para estandarizarse. No definen la metodología; ejecutan una
> tarea dentro de ella.

## Previstas

- diagnóstico de empresa;
- ficha de proceso (basada en el [modelo operativo](../modelo-operativo.md));
- ADR (Architecture Decision Record);
- definición de entidad de negocio;
- definición de automatización;
- definición de agente de IA;
- bitácora de avance.

_Aún sin desarrollar._
