# Plantillas

Plantillas reutilizables para ejecutar tareas repetibles de la metodología.

> Las plantillas no definen la metodología; ejecutan una tarea dentro de ella.

## Disponibles

- [Plantilla de diagnóstico](./plantilla-diagnostico.md) — guion y entregable de
  la Fase 0.
- [Plantilla de roadmap](./plantilla-roadmap.md) — el puente entre el diagnóstico
  y el programa; deriva la propuesta comercial.
- [Plantilla de propuesta](./plantilla-propuesta.md) — documento comercial para
  cerrar la Fase 0 con la empresa.
- [Ficha de proceso](./plantilla-proceso.md) — modela un proceso según el
  [modelo operativo](../modelo-operativo.md).
- [Plantilla de ADR](./plantilla-adr.md) — registra el porqué de una decisión.

## Previstas

- definición de entidad de negocio;
- definición de automatización;
- definición de agente de IA;
- bitácora de avance.
