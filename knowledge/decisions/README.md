# Decisions (ADRs)

El *porqué* de las decisiones grandes del proyecto, con sus alternativas.

Formato: [../../metodo/plantillas/plantilla-adr.md](../../metodo/plantillas/plantilla-adr.md).
No se editan retroactivamente; si una decisión cambia, se agrega un ADR nuevo que
la supersede.
